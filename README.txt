Instructions: v0.3
1. Install BandagedBD.
2. Open Discord.
3. Navigate To User Settings.
4. Scroll All The Way Down To Plugins/Themes.
5. Click The Button On The Top To Open Plugins/Themes Folders.
6. Copy Discord Serenity Plugins/Themes To Folders.
7. Enable The DiscordCrypt Plugin And Restart Discord.
8. Once Restarted Create Your DiscordCrypt Password.
9. Navigate Back To Plugins/Themes.
10. Enable All Plugins And The Official Serenity Theme.
